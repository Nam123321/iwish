2. Why it exists

The Problem: When cloning website templates or pages, visual elements (like custom logos, icons, illustrations, and background SVG patterns) often point to absolute URLs hosted on the source domain. When that domain goes down, redirects, or blocks the scraper, the cloned layout breaks.

The Solution: GoClone runs an asynchronous crawler that resolves and downloads all static assets (stylesheets, JavaScript files, images, SVGs) into local subfolders (css/, js/, imgs/). It then parses the HTML and rewrites all references to point to these relative local assets, creating a standalone, offline-ready copy of the page.

5. Upgrade Blueprint: The Hybrid Cloner

By merging GoClone's asset-harvesting with Hallmark's semantic Visual DNA study, we build a far superior website cloner for I-Wish:

1. GoClone Track (Asset Scraper)

Crawl the target URL to extract and download all high-value visual assets (brand logos, custom icons, background illustrations) into a local imgs/ folder.

2. Hallmark Track (Visual DNA)

Perform a semantic visual audit on the reference site. Extract macro structures, layout skeletons, font pairing roles, and HSL color anchors instead of copying raw messy styles.

3. Re-linked Synthesis

Assemble the new component using clean responsive Tailwind CSS tokens, and dynamically insert references to the downloaded local assets (logos and icons) to preserve brand styling.

6. Constraints & Edge Cases

Go Runtime Dependency: Since I-Wish is a Node-based runtime, executing a compiled Go binary introduces cross-compilation and setup hurdles. We address this by porting the GoQuery and Colly logic directly to Node.js using `cheerio` and `axios/fetch` crawlers.

Dynamic SPAs: Standard HTTP GET fetches return empty HTML roots for client-side React apps. In these situations, I-Wish fallbacks to screenshot vision passes to analyze the design.

7. Example: Link Re-writing

Below is how the crawler rewrites resource links from absolute source URLs to relative local folders:

// Original fetched HTML
<link rel="stylesheet" href="https://stripe.com/assets/global.css">
<img src="https://stripe.com/assets/logo.svg">

// Rewritten HTML using Cheerio (GoQuery Port)
<link rel="stylesheet" href="css/global.css">
<img src="imgs/logo.svg">
