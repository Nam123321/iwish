Headroom Integration Guide (Adoption Review Pack)

1. Classification & Placement

  • Shape: SYSTEM_SKILL (Bundle)
  • Role: supportive
  • Framework Placement: Middleware / background proxy layer between I-Wish agents and the LLM API provider.

2. Core Use Cases

  • Tự động nén payload (JSON, AST, RAG context) giảm 60-95% token.
  • Cung cấp proxy drop-in không cần sửa code cốt lõi của I-Wish agents.

3. Constraints & Edge Cases

  • Over-compression: Có thể làm mất stack trace quan trọng.
  • Latency: Tăng thời gian phản hồi do overhead phân tích AST/JSON.
  • Yêu cầu hệ thống: Python 3.10+.

4. Orch Routing Hints

  • Kích hoạt Headroom khi xử lý các file codebase cực lớn (như trong Repo Absorption hoặc Analyze Codebase).
  • Đề xuất dùng khi Dev-Agent báo lỗi vượt quá Context Window.
