turbovec

Memory-efficient vector search using TurboQuant quantization algorithm

Shape: skill Role: supportive Status: Draft Registered Maturity: Production-ready
Why Adopt? / Tại sao tích hợp?
  • 16x memory compression (2-bit or 4-bit config).
  • Data-oblivious: no offline codebook training required.
  • SIMD acceleration (NEON/AVX-512BW) on local CPUs.
  • Out-of-the-box support for popular RAG frameworks.
Exclusions / Giới hạn & Điểm lưu ý
  • Max-Marginal Relevance (MMR) search is NOT supported.
  • Only Cosine distance is supported (vectors normalized at entry).
  • Bus Factor = 1: entirely maintained by a single developer.
  • Sub-byte deletions shift elements internally.
🚀 Code Usage Examples / Ví dụ sử dụng

Using TurboQuantVectorDb with Agno RAG:

from agno.knowledge.embedder.openai import OpenAIEmbedder
from turbovec.agno import TurboQuantVectorDb

# Dimensions must match OpenAI embedder (1536)
vector_db = TurboQuantVectorDb(
    embedder=OpenAIEmbedder(),
    bit_width=4,
    path="./memory"
)
vector_db.create()

Using TurboQuantVectorStore in LangChain:

from turbovec.langchain import TurboQuantVectorStore
from langchain_openai import OpenAIEmbeddings

embeddings = OpenAIEmbeddings()
vector_store = TurboQuantVectorStore(
    embedding=embeddings,
    bit_width=4
)

Using TurboQuantVectorStore in LlamaIndex:

from turbovec.llama_index import TurboQuantVectorStore

vector_store = TurboQuantVectorStore(
    dim=1536,
    bit_width=4,
    persist_path="./storage"
)

Using TurboQuantIndex in Rust core:

use turbovec::TurboQuantIndex;

// 1536-dim vectors compressed to 4 bits
let mut index = TurboQuantIndex::new(1536, 4).unwrap();
index.add(&vectors);
let results = index.search(&queries, 10);

Issue #89 Fix: Transactional Upsert (Validate First, Delete Second)

def upsert_safe(self, content_hash: str, documents: List[Document]):
    # 1. Embed and pre-validate embeddings first
    self._embed_missing(documents)
    missing = [d for d in documents if not d.embedding]
    if missing:
        raise ValueError("Missing embeddings")
    
    # Check shapes
    for doc in documents:
        qvec = np.asarray(doc.embedding, dtype=np.float32)
        if qvec.shape[-1] != self.dimensions:
            raise ValueError(f"Dim mismatch: got {qvec.shape[-1]}, expected {self.dimensions}")
            
    # 2. Deletions happen safely only after validation passes
    if self.content_hash_exists(content_hash):
        self._delete_by_content_hash(content_hash)
        
    # 3. Final insert
    self.insert_validated(content_hash, documents)

Issue #90 Fix: Intra-batch Duplicate ID Guard

def check_duplicate_ids(self, documents: List[Document]):
    seen = set()
    for doc in documents:
        if not doc.id:
            continue
        if doc.id in seen:
            raise ValueError(f"Duplicate document ID '{doc.id}' detected in the same batch")
        seen.add(doc.id)
