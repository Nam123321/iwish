⚠️ 6. Constraints & Safety Gates

  • Self-Healing Threshold: Limit automated compilation repair loops to exactly 3 retries to prevent infinite token depletion.
  • AST XML Parser Mismatch: Regrade to standard regex blocks if the LLM output tags are broken.
  • No Dynamic Eval: Never execute raw eval(). Parse structures strictly with ast.literal_eval() or Pydantic.
  • Workspace Isolation: Restrict npm global installs; keep tool environments local to the workspace.
