Integration Guide for antigravity-awesome-skills
State: Draft Review

01. Overview & Placement

What is it?

A reference module pack aggregating over 1,470+ unique agentic skills (4,570+ file instances) from official and community sources, equipped with path-filtering and safety checkers.

Placement

Classified as a supportive-registry. Placed primarily in the orchestration and validate stages to assist with system-wide skill validation and RAG-based context lookup.

02. Core Use Cases

Adopt: Skill Quality Linting (SYSTEM_SKILL)

Integrate their schema validation and frontmatter checker scripts directly into I-Wish's /create-skill workflow. This ensures any custom skill generated meets strict formatting standards.

Selective: Specialized Playbooks

Deploy outstanding developer guidelines (e.g., andrej-karpathy-skills) to enhance dev-agent coding accuracy and testing discipline.

03. Risks & Constraints

⚠️ Token & Context Poisoning
Loading all 1,470+ skills directly into the agent folder will immediately exhaust LLM context windows (Gemini Issue #215). Strict selective extraction is mandatory.
❌ Global Installer Execution
The CLI installer scripts write files outside the workspace (e.g. ~/.agents). To maintain sandbox isolation, global installer hooks must be SKIPPED.

04. Actionable Decisions for User
