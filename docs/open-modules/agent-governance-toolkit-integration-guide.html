Adoption Review Pack

Integration Guide & Route Specifications for microsoft/agent-governance-toolkit

Awaiting Promotion Shape: skill-attachment / skill Role: supportive / foundational

1. What is it / Tổng quan

Bản hướng dẫn tích hợp dành cho Module mở rộng hấp thụ chọn lọc từ microsoft/agent-governance-toolkit.

  • Current State: Draft assets prepared under _iwish-output/iwish-skills/
  • Extracted Components: Cedar/Rego Policy Evaluators, Cost Budget Monitors, and recursive Credential Redactor.
  • Skipped Components: Soft Python Sandbox (frame breakout vulnerability), VS Code Webviews, and multi-language FFI bindings (.NET, Rust, Go).

2. Why it exists / Lý do áp dụng

Để ngăn chặn các rủi ro bảo mật thuộc **OWASP Agentic AI Top 10** trong hệ thống I-Wish:

  • Gap Filled: Bổ sung khả năng kiểm duyệt tự động hành vi của Agent (ABAC/RBAC) và cắt giảm chi phí (Auto-throttle cost) khi Agent bị lặp vô hạn.
  • PII/CRI Safety: Tự động làm sạch các khoá API nhạy cảm trước khi lưu vết logs.

3. Input -> Process -> Output / Luồng xử lý

Input: Tên tool gọi, tham số gọi (payload), và nhãn nhạy cảm (context).

Process: Chạy qua bộ kiểm duyệt Cedar, rà quét regex khoá bảo mật, và đối chiếu hạn mức chi phí còn lại.

Output: Quyết định thực thi (allow / deny / warn / transform) cùng payload đã được làm sạch.

4. Use cases / Trường hợp sử dụng

Nên dùng / Core Use Cases

  • Redact OpenAI, AWS, và JWT keys khỏi logs.
  • Giới hạn số lượng tool call tối đa cho mỗi tác vụ phụ của subagent.

Không nên dùng / Do-Not-Use

  • Không chạy các dòng mã nguy hiểm thông qua Python soft-sandbox.
  • Không biên dịch chéo các SDK đa ngôn ngữ khác.

5. Risks & Stress Cases / Hạn chế & Rủi ro

  • Regex Backtracking (ReDoS): Quét các văn bản quá lớn (>10MB) bằng regex phức tạp có thể gây treo CPU. Biện pháp: Cài đặt timeout cố định cho tiến trình quét.
  • SSRF Bypass: Agent sử dụng URL redirect để truy cập IP local. Biện pháp: Phân giải DNS và kiểm tra IP đích trước khi gửi request.

⚠️ Lưu ý: Toàn bộ tài sản tích hợp phải tuân thủ nghiêm ngặt tính cô lập dự án, không ghi đè vào thư mục chung của hệ thống.
