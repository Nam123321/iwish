Open Module Integration Guide: spec-kit

This integration guide defines the strategy for adopting, adapting, or referencing components of the spec-kit repository into the I-Wish ecosystem.

System Placement: Ported components will reside in .agent/skills/editor-compatibility/ as a SYSTEM_SKILL.

1. Classification & System Placement

  • Shape & Role: Utility and integration adapter (SYSTEM_SKILL path).
  • Core Role: Translates and registers I-Wish behavioral tokens, checklists, and guides into agent directories of third-party IDE editors (e.g. Claude Code, Copilot, Goose, Roo).

2. Core Use Cases

  • Multi-IDE Workspace Compatibility: Allows users working in editors like Claude Code or Cursor (Roo) to export and run I-Wish checklists and prompt configurations natively.
  • Path Integrity Shield: Uses the path traversal guard helper (_ensure_inside) to secure all local files writes against directory escape vulnerabilities.

3. Operational Constraints

  • Scope Limit: Do not adopt the shell runner engine or templates from spec-kit. I-Wish has a more mature workflow and verification engine.
  • Python Dependency: The ported utilities must be written in Python to integrate natively with I-Wish backend services.

4. Agent & Workflow Coordination

  • orch-agent Routing Hints: If the user specifies working in Cursor, Windsurf, or Copilot, orch-agent should offer to execute skill publishes.
  • dev-agent Rules: Must validate all file write commands through the _ensure_inside path validator.

5. YAML Frontmatter Specification for Ported Skills

name: iwish-path-security description: Normalizes and verifies that file operations remain inside the workspace. inputs: [target_path, base_path] outputs: [is_safe, normalized_path] mcp_tools_required: [] subagent_triggers: []

6. Review Questions for the User

  1. Do you actively use editors like Roo, Copilot, or Goose alongside I-Wish, and do you want I-Wish skills exported to them?
  2. Should we proceed with creating the editor-compatibility skill to port the path traversal shield and format conversion helpers?
