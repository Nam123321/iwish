# Brand ID & Brand Guideline SKILL / Workflow Package

This package contains a reusable workflow for creating professional brand identity and brand guideline systems.

## Files

- `SKILL.md` — Full workflow and production standard.
- `questionnaire.md` — Complete client intake questionnaire.
- `prompt_library.md` — Reusable prompts for brand boards, guideline, ads, POSM, brochure, website hero, email, line-art logo.
- `production_checklist.md` — Final QA checklist.

## How to Use

1. Start with `questionnaire.md`.
2. Confirm brand strategy and required deliverables.
3. Use `SKILL.md` to structure the production workflow.
4. Use `prompt_library.md` to create specific visual outputs.
5. Use `production_checklist.md` before final delivery.

## Recommended Output

For professional projects, deliver:
- Brand guideline HTML or PDF
- Logo assets
- Icon set
- Design tokens
- UI mockups
- Marketing templates
- Organized ZIP package
