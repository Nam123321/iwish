# UX Principles & Design Alignment

1. **Visual Consistency**: Adhere strictly to the established design system tokens (colors, typography, spacing). Do not invent ad-hoc styles.
2. **Accessibility (a11y)**: Ensure all interactive elements have proper ARIA attributes, semantic HTML, and keyboard navigability.
3. **Spacing and Layout**: Use predictable scale systems for margins and padding. Ensure responsive behavior across different viewport sizes.
4. **DOM-Driven Layout**: Implement components using the DOM-driven layout derived from the Stitch UI visual contract, rather than relying on backend schema-driven layouts for UI structural decisions.
