# /gen-dashboard

Lệnh này sẽ tự động tổng hợp toàn bộ tài liệu trong thư mục `docs/`, các fragments, workflows và kiến thức hiện có của Iwish để tạo ra trang Dashboard tương tác và tài liệu User Guide dưới dạng HTML (`_iwish-output/index.html` hoặc tương tự).

## Cách thực hiện:
Chạy lệnh CLI sau trên Terminal:
```bash
iwish gen-dashboard
```
