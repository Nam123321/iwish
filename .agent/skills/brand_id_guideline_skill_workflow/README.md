# Brand ID & Brand Guideline SKILL / Workflow Package (Enhanced)

This package contains a production-ready, reusable workflow skill for designing or refactoring professional brand identities, logo systems, and design tokens.

## What's New in This Version
1.  **Kapferer's Brand Prism Integration**: The brand strategy is structured around the 6 pillars of Kapferer's Prism to ensure deep conceptual alignment.
2.  **5-Option Logo Brainstorming**: Models must brainstorm at least 5 distinct logo concepts and explain their geometric parts, meaning, and typographic pairing in detail.
3.  **Strict Sequential Approval Blocker**: A hard stop is enforced at Phase 4. Logo systems must be approved and locked by the client before expanding the design system or touchpoints.
4.  **Logo & Brand ID Refactoring Workflow**: Added three dedicated paths (Path A: Cleanup, Path B: Evolution, Path C: Revolution) for modernizing and fixing existing user assets.
5.  **Modern Tools Recommendations**: Integrated guidance on tools like Miro, Figma, Recraft.ai, Vectorizer.ai, Style Dictionary, and Brandpad.

1.  Initiate project discovery using the [questionnaire.md](file://{project-root}/scratch/brand_id_guideline_skill_workflow/brand_id_guideline_skill_workflow/questionnaire.md).
